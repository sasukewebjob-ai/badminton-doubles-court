# プロジェクト概要
バドミントンのダブルス試合で、各節のコート割（ペア・対戦相手・休み）を自動生成するWebアプリ。

## 目的・ゴール
バドミントンクラブ等で、ダブルスマッチを公平に組む。
- 節数を 10 / 15 / 20 / 25 / 30 から選択可
- 休み回数を均等化（差は最大1回を保証）
- 同じペア・同じ対戦の偏りを最小化（150試行のランダム最適化）
- スマホで誰でもすぐ使える

## 技術スタック
- HTML / CSS / JavaScript（単一ファイル、外部依存なし）
- モバイル端末（iPhone/Android）での利用を想定
- ライブラリ・フレームワーク未使用

## やらないこと・制約
